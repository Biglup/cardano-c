UPLC
==========================

Untyped Plutus Core (UPLC) is the on-chain execution language of Cardano Plutus
scripts. This section covers the public UPLC utilities exposed by the library.
Phase-2 transaction evaluation, which runs scripts on the in-process virtual
machine, is provided by the native transaction evaluator documented under the
Transaction Builder.
