Scripts
========

This section of the documentation provides an in-depth look at the various script types utilized within the Cardano ecosystem. Scripts are fundamental components in Cardano's smart contract infrastructure, enabling the execution of complex logic in a decentralized manner. The Plutus scripting language, along with its data types, allows for the creation of robust and secure smart contracts and decentralized applications (dApps).
